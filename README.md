# E-Commerce Backend API

## Overview

A backend e-commerce application developed using Spring Boot, Spring Security, JWT Authentication, Spring Data JPA, and MySQL.

## Features

* User Registration
* User Login
* JWT Authentication & Authorization
* Product Management
* Category Management
* Order Processing
* Role-Based Access Control (Admin/User)
* RESTful APIs
* Exception Handling
* Input Validation

## Technologies Used

* Java
* Spring Boot
* Spring Security
* JWT
* Spring Data JPA
* MySQL
* Maven
* Postman

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* Security Layer

## How to Run

1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run the Spring Boot application
5. Test APIs using Postman
